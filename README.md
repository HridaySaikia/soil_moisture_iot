# 🌱 Soil Moisture IoT Dashboard

A real-time IoT-based soil moisture monitoring system using ESP32 and a modern web dashboard.

---

## 🚀 Features

- 📡 Real-time moisture monitoring
- 📊 Interactive charts & analytics
- 🔔 Threshold-based alerts
- 🔐 Authentication using Clerk
- 🌐 Cloud-connected IoT system

---

## 🛠 Tech Stack

- Frontend: Next.js, Tailwind CSS
- Backend: API Routes
- Database: MongoDB
- IoT: ESP32
- Charts: Recharts
- Auth: Clerk
